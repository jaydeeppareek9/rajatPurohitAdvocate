## 🚀 How to Host FREE on GitHub Pages (Step by Step)

### Step 1 — Create a GitHub Account
Go to **github.com** → Sign Up (free). Use any email.

### Step 2 — Create a New Repository
1. Click the **"+"** icon (top right) → **New repository**
2. Name it exactly: `rajat-purohit` (or any name you like)
3. Set it to **Public**
4. Click **"Create repository"**

### Step 3 — Upload the Files
1. In your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop the `index.html` file from this folder
3. Click **"Commit changes"** (green button)

### Step 4 — Enable GitHub Pages
1. Click **"Settings"** tab in your repository
2. Scroll down to **"Pages"** (left sidebar)
3. Under **"Source"**, select **"Deploy from a branch"**
4. Choose **"main"** branch → **"/ (root)"**
5. Click **Save**

### Step 5 — Get Your Free URL
- Wait 2–3 minutes ⏱️
- Your website will be live at:
  ```
  https://YOUR-USERNAME.github.io/rajat-purohit/
  ```
  Example: `https://rajatpurohit.github.io/rajat-purohit/`

### Step 6 — Use URL in QR Code
1. Copy your GitHub Pages URL
2. Go to **qr-code-generator.com** or **qrcode-monkey.com**
3. Paste your URL → Customize with gold color → Download
4. Use this QR code on your visiting card!

## ✏️ How to Add Your Photo

1. Open `index.html` in Notepad or any text editor
2. Find this section:
   ```html
   <div class="photo-placeholder">
   ```
3. Replace it with:
   ```html
   <img src="your-photo.jpg" alt="Rajat Purohit" style="width:100%;height:100%;object-fit:cover;" />
   ```
4. Upload your photo file alongside `index.html` on GitHub
